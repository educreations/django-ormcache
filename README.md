# Django ORMCache

A cache manager mixin that provides some caching of objects for the ORM.

https://pypi.org/project/django-ormcache/


# Installation / Setup / Usage

```
pip install django-ormcache
```


# Testing

Run the tests with:

```
pip install -e .[tests]
tox
```
